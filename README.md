# Pete Rooke 👋

**Technology strategist & builder — Brisbane, Australia 🇦🇺**

A decade at the intersection of **AI, infrastructure, and higher education**. I'm interested in how AI *actually* gets adopted in organisations — not the demos, but the messy reality of integration, change management, and making it stick.

When I'm not thinking about that, I'm building things to scratch my own itches — mostly self-hosted, mostly wired up to Claude.

### 🤖 Building with AI

| Project | What it does |
| --- | --- |
| **[launchpad](https://github.com/PeteInBrissie/launchpad)** | Self-hosted ADHD school-management tool — a web app paired with an e-paper display, built to make school-day logistics calmer. `TypeScript` |
| **[japanesetutor](https://github.com/PeteInBrissie/japanesetutor)** | iOS app for learning Japanese from beginner to conversational — SwiftUI + Claude + spaced-repetition flashcards. `Swift` |
| **[MeatSpace](https://github.com/PeteInBrissie/MeatSpace)** | Log daily protein intake in plain language; Claude Haiku parses each meal and optionally writes to Apple Health. Wabi-sabi design — aged indigo on warm linen. `Swift` |
